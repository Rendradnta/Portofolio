# Resa4You - Vite React Portfolio

## Run locally

1. Install deps:
   ```bash
   npm install
   ```
2. Run dev server:
   ```bash
   npm run dev
   ```

This project uses:
- Vite + React
- TailwindCSS
- framer-motion
- react-tsparticles (no loadFull to maximize compatibility)
- react-icons

Dummy images are included in the project root for quick testing.
